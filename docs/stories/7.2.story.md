## Story

As an **operations user in Brazil**, I want to **usar toda a interface em português brasileiro e poder criar webhooks diretamente pelo formulário**, so that **eu entenda todas as ações sem barreira de idioma e tenha funcionalidade completa de webhooks**.

## Acceptance Criteria

### Padronização PT-BR

- [x] Toda a interface está em português brasileiro — nenhum label, botão, título, placeholder, mensagem de erro ou estado vazio em inglês.
- [x] Termos técnicos mantidos em inglês: "webhook", "API", "DLQ", "rate limit", "circuit breaker", "Slack", "Discord".
- [x] Padrão de botões consistente: "Salvar", "Cancelar", "Excluir", "Editar", "Criar", "Filtrar", "Limpar".
- [x] Padrão de estados: "Carregando...", "Salvando...", "Erro ao carregar...", "Nenhum item encontrado".
- [x] Páginas de login e signup traduzidas: "Entrar", "Criar conta", "E-mail", "Senha".

### Formulário de Webhook

- [x] Modal de criação de webhook exibe formulário completo no lugar do placeholder atual.
- [x] Formulário contém: URL (obrigatório), Descrição (opcional), Habilitado (toggle, default true).
- [x] Validação: URL é obrigatório e deve ser formato válido de URL.
- [x] Ao submeter: chama `webhookService.createWebhook()`, exibe loading no botão, fecha modal e recarrega lista.
- [x] Erros de API exibidos inline no formulário.

## Technical Context for Dev Agent

### Padrão de tradução (sem i18n)

Transcrever diretamente nos componentes. Exemplo:

```tsx
// ANTES:
<th>URL</th>
<th>Status</th>
<th>Success Rate</th>
<button>New Webhook</button>

// DEPOIS:
<th>URL</th>
<th>Status</th>
<th>Taxa de Sucesso</th>
<button>Novo Webhook</button>
```

### Formulário de Webhook — Integração

O service já existe e funciona:

```typescript
// webhook.service.ts
webhookService.createWebhook({ url: string, description?: string, enabled?: boolean })
```

Padrão de formulário a seguir (mesmo padrão de `TaskModal.tsx`):
1. Usar `FormInput` de `components/composite/FormField`.
2. Usar `Button` de `components/ui/Button` com variant `primary`.
3. Estado de loading durante submit.
4. Feedback de erro inline.
5. Após sucesso: fechar modal + chamar `loadWebhooks()` para recarregar lista.

### Confirmação de exclusão

Substituir `confirm('Delete this webhook?')` por `confirm('Excluir este webhook? Esta ação não pode ser desfeita.')`.
