## Story

As an **operations user**, I want to **visualizar o workspace de fluxos com cards e colunas mais densos, legiveis e operacionais**, so that **o quadro transmita a mesma clareza e maturidade visual dos screenshots de referencia sem perder comportamento atual**.

---

## Acceptance Criteria

- [x] O workspace de fluxos em board mode usa uma linguagem visual coerente com a fundacao e o shell refinados.
- [x] Cabecalhos de coluna comunicam melhor volume, estado e contexto operacional.
- [x] Cards exibem metadata principal com hierarquia mais clara: titulo, time, tags, progresso e estado, sem esconder a informacao principal necessaria para triagem visual no board.
- [x] Estados ativo, bloqueado, concluido e inativo possuem diferenciacao visual clara.
- [x] Os modos atuais do workspace (quadro/tabela/indicadores) continuam funcionando sem regressao.
- [x] Drag-and-drop continua funcional e compreensivel apos o refinamento visual.
- [x] Estados de loading, empty e error do workspace e das colunas vazias continuam explicitos apos o refinamento visual.
- [x] Existem evidencias before/after do board em desktop e mobile.
