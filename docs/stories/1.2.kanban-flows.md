# Story 1.2: Fluxos Kanban com Drag & Drop

**Epic:** 1.0 (MVP Foundation)
**Status:** Done
**Priority:** P1
**Points:** 8

## Story
Como usuário, preciso de um quadro Kanban customizável com colunas que posso arrastar e soltar cards para gerenciar o fluxo de trabalho.

## Acceptance Criteria

### AC-1: Kanban Board Display
- [x] Exibe colunas customizáveis (Backlog, A Fazer, Em Progresso, Finalizado)
- [x] Backlog sempre oculto, Finalizado fixo
- [x] Permite criar/editar nome de colunas (exceto Backlog e Finalizado)
- [x] Mostra total de cards por coluna
- [x] Design: white background, black text, minimal

### AC-2: Drag & Drop
- [x] Permite arrastar card entre colunas
- [x] Permite reordenar cards na mesma coluna
- [x] Persiste ordem ao backend
- [x] Suporta touch (mobile)

### AC-3: API Endpoints
- [x] POST /api/flows - Criar novo flow
- [x] GET /api/flows/:id - Obter flow com colunas
- [x] PUT /api/flows/:id/columns - Atualizar coluna
- [x] POST /api/flows/:id/columns - Adicionar coluna

### AC-4: Database
- [x] Tabela `flows` (nome, user_id)
- [x] Tabela `columns` (name, order, flow_id)
