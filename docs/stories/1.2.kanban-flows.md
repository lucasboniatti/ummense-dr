# Story 1.2: Fluxos Kanban com Drag & Drop

**Epic:** 1.0 (MVP Foundation)
**Status:** Ready for Review
**Priority:** P1
**Points:** 8

## Story
Como usuário, preciso de um quadro Kanban customizável com colunas que posso arrastar e soltar cards para gerenciar o fluxo de trabalho.

## Acceptance Criteria

### AC-1: Kanban Board Display
- [ ] Exibe colunas customizáveis (Backlog, A Fazer, Em Progresso, Finalizado)
- [ ] Backlog sempre oculto, Finalizado fixo
- [ ] Permite criar/editar nome de colunas (exceto Backlog e Finalizado)
- [ ] Mostra total de cards por coluna
- [ ] Design: white background, black text, minimal

### AC-2: Drag & Drop
- [ ] Permite arrastar card entre colunas
- [ ] Permite reordenar cards na mesma coluna
- [ ] Persiste ordem ao backend
- [ ] Suporta touch (mobile)

### AC-3: API Endpoints
- [ ] POST /api/flows - Criar novo flow
- [ ] GET /api/flows/:id - Obter flow com colunas
- [ ] PUT /api/flows/:id/columns - Atualizar coluna
- [ ] POST /api/flows/:id/columns - Adicionar coluna

### AC-4: Database
- [ ] Tabela `flows` (nome, user_id)
- [ ] Tabela `columns` (name, order, flow_id)
