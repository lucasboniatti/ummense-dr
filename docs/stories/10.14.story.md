## Story

As a **user**, I want to **buscar rapidamente tarefas, fluxos e webhooks usando Cmd+K**, so that **eu possa navegar e encontrar informações sem precisar percorrer menus**.

---

## Acceptance Criteria

- [ ] Modal de busca abre com Cmd+K (Mac) ou Ctrl+K (Windows).
- [ ] Campo de busca com foco automático ao abrir.
- [ ] Resultados agrupados por tipo: Tarefas, Fluxos, Webhooks.
- [ ] Navegação por teclado (Arrow keys, Enter, Escape).
- [ ] Click em resultado navega para a página correta.
- [ ] Busca debounce para performance.
- [ ] Histórico de buscas recentes.
- [ ] Dark mode suportado.
- [ ] Primeira entrega funciona em modo frontend-first; novo endpoint backend não é obrigatório para concluir a story.

## Dev Notes

### Relevant Context

- `AppTopbar.tsx` já é o ponto natural de entrada para busca global no shell.
- A story deve começar com busca client-side sobre superfícies já presentes no produto, evitando expandir backend sem necessidade comprovada.
- Resultados iniciais devem cobrir tarefas, fluxos e webhooks visíveis na operação atual.

### API Design

```tsx
import { GlobalSearch } from '@/components/GlobalSearch';

// In AppTopbar
<GlobalSearch />

// Or with trigger button
<Button onClick={() => setSearchOpen(true)}>
  <Search className="w-4 h-4" />
  <span className="ml-2">Buscar...</span>
  <kbd className="ml-auto">⌘K</kbd>
</Button>
```

### Command Component (cmdk)

```tsx
import { Command } from 'cmdk';

<Command.Dialog open={open} onOpenChange={setOpen}>
  <Command.Input placeholder="Buscar tarefas, fluxos, webhooks..." />
  <Command.List>
    <Command.Empty>Nenhum resultado encontrado.</Command.Empty>

    <Command.Group heading="Tarefas">
      {tasks.map(task => (
        <Command.Item key={task.id} onSelect={() => navigate(`/cards/${task.cardId}`)}>
          {task.title}
        </Command.Item>
      ))}
    </Command.Group>

    <Command.Group heading="Fluxos">
      {flows.map(flow => (
        <Command.Item key={flow.id} onSelect={() => navigate(`/dashboard/automations/${flow.id}`)}>
          {flow.name}
        </Command.Item>
      ))}
    </Command.Group>
  </Command.List>
</Command.Dialog>
```
