## Dev Notes

### Relevant Context

- As micro-interações desta story devem se apoiar no sistema de animações da `10.17`.
- O foco principal está nos primitives compartilhados e nas superfícies operacionais mais frequentes.
- Nenhuma micro-interação deve degradar acessibilidade ou ignorar `prefers-reduced-motion`.

### Hover Lift Effect

```css
.card-hover {
  transition: transform 0.2s ease, box-shadow 0.2s ease;
}

.card-hover:hover {
  transform: translateY(-2px);
  box-shadow: 0 8px 25px -5px rgba(0, 0, 0, 0.1);
}
```

### Button Click Feedback

```css
.btn-press {
  transition: transform 0.1s ease;
}

.btn-press:active {
  transform: scale(0.98);
}
```

### Focus Ring Animation

```css
.focus-ring:focus-visible {
  outline: none;
  box-shadow: 0 0 0 3px rgba(59, 130, 246, 0.5);
  animation: focus-ring-pulse 1s ease infinite;
}

@keyframes focus-ring-pulse {
  0%, 100% { box-shadow: 0 0 0 3px rgba(59, 130, 246, 0.5); }
  50% { box-shadow: 0 0 0 5px rgba(59, 130, 246, 0.3); }
}
```

### Link Underline Animation

```css
.link-underline {
  position: relative;
}

.link-underline::after {
  content: '';
  position: absolute;
  bottom: 0;
  left: 0;
  width: 0;
  height: 1px;
  background: currentColor;
  transition: width 0.2s ease;
}

.link-underline:hover::after {
  width: 100%;
}
```
