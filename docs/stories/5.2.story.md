## Story

As a **release owner**, I want to **replace scaffold quality checks and harden runtime security defaults**, so that **cloud/local releases are validated by real gates and safer defaults**.

---

## Acceptance Criteria

- [x] `lint`, `typecheck`, `test` executam validações reais (não scaffold echo).
- [x] Frontend build não ignora erros de type/lint.
- [x] Backend CORS deixa de ser permissivo global e passa a usar allowlist.
- [x] Autenticação WebSocket valida assinatura JWT (sem parse inseguro de payload).
- [x] CI workflow versionado no repositório com sequência `lint -> typecheck -> test -> build`.

## Tasks / Subtasks

- [x] Criar configuração de lint real no monorepo (`eslint` + flat config).
- [x] Atualizar scripts de workspaces para checks reais.

- [x] Adicionar testes reais mínimos para segurança/config.
