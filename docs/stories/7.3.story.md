## Story

As an **operations user**, I want to **ver estados consistentes de carregamento, telas vazias com orientação e mensagens de feedback elegantes**, so that **eu sempre saiba o que está acontecendo no sistema e nunca fique confuso com telas genéricas**.

## Technical Context for Dev Agent

### EmptyState — Interface

```typescript
interface EmptyStateProps {
  icon?: React.ReactNode;          // Ícone Lucide (ex: <Inbox size={48} />)
  title: string;                   // Ex: "Nenhum fluxo encontrado"
  description?: string;            // Ex: "Crie seu primeiro fluxo para..."
  actionLabel?: string;            // Ex: "Criar Fluxo"
  onAction?: () => void;           // Callback do botão CTA
  variant?: 'default' | 'compact'; // default = full page, compact = inline
}
```

**Padrão visual:**
- Centralizado, ícone em `text-neutral-300` tamanho 48px
- Título: `text-neutral-700 font-semibold text-lg`
- Descrição: `text-neutral-500 text-sm`
- Botão: `<Button variant="primary">`

### Spinner — Interface

```typescript
interface SpinnerProps {
  size?: 'sm' | 'md' | 'lg';  // 16px, 24px, 32px
  className?: string;
}
// CSS: animate-spin rounded-full border-2 border-neutral-200 border-t-primary-600
```

### PageLoader — Interface

```typescript
interface PageLoaderProps {
  message?: string;  // Ex: "Carregando configurações..."
}
// Renders: centered Spinner (lg) + message text
```

### Toast — Interface

```typescript
interface Toast {
  id: string;
  type: 'success' | 'error' | 'info' | 'warning';
  title: string;
  description?: string;
  duration?: number; // ms, default 4000
}

interface ToastContextType {
  toasts: Toast[];
  addToast: (toast: Omit<Toast, 'id'>) => void;
  removeToast: (id: string) => void;
}

// Shortcut hook:
// const toast = useToast();
// toast.success('Salvo com sucesso!');
// toast.error('Falha ao atualizar');
```

**Padrão visual do Toast:**
- Posição: fixed top-4 right-4, z-50
- Background por type: success=green, error=red, info=blue, warning=yellow
- Animação: slide-in da direita, fade-out ao expirar
- Tamanho: max-w-sm, rounded-lg, shadow-lg

### Integração no _app.tsx (após WP-1)

```tsx
<AuthProvider>
  <ToastProvider>
    <AuthGuard>
      <AppShell>
        <Component {...pageProps} />
      </AppShell>
    </AuthGuard>

  </ToastProvider>
</AuthProvider>
```
