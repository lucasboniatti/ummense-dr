## Story

As a **user**, I want to **alternar entre modo claro e escuro através de um toggle no sidebar**, so that **eu possa escolher a aparência que me agrada e o sistema respeite minha preferência do sistema operacional**.

---

## Acceptance Criteria

- [x] Toggle de tema visível no footer do sidebar.
- [x] Toggle exibe ícone de sol (light) ou lua (dark) conforme tema atual.
- [x] Clique no toggle alterna entre light e dark instantaneamente.
- [x] Tema é persistido no localStorage e mantido ao recarregar.
- [x] Sistema respeita preferência inicial do sistema operacional (`prefers-color-scheme`).
- [x] Transição entre temas é suave (fade transition).
- [ ] Todos os componentes existentes funcionam corretamente em ambos os temas.
