## Story

As a **platform operator**, I want to **receive real-time execution history updates via WebSocket**, so that **I can monitor automation execution in live time without manual page refresh**.

---

## Acceptance Criteria

- [x] **[CRITICAL]** WebSocket server implementation with connection management
  - Server listens on dedicated port (9001, configurable via ENV)
  - Clients subscribe to user's execution_history channel (user_id-based)
  - Server broadcasts execution updates every 5-10s (configurable via ENV)
  - Handle client disconnects gracefully (automatic reconnect after 3s on client side)
  - No memory leaks (validated via 1-hour sustained load test with 100+ concurrent connections)
  - Load test: 100+ concurrent WebSocket connections must sustain without degradation

- [x] **[HIGH]** Real-time latency and performance targets
  - Latency: <500ms from database write to client notification (measured via instrumentation)
  - Delta detection: Only changed execution records sent (no full history replay)
  - Message size: <1KB per delta (compression if needed)
  - CPU usage: <2% per 50 concurrent connections (baseline)

- [x] **[HIGH]** Redis pub/sub integration for horizontal scaling
  - Multi-server deployments supported via Redis pub/sub
  - Single server fallback: In-memory subscription (no Redis required for dev)
  - Pub/sub channel naming: `execution-updates:{user_id}`
  - Message format: JSON with execution_id, changed_fields, timestamp

- [x] **[MEDIUM]** Connection lifecycle management
  - Graceful shutdown: Drain existing connections (max 30s timeout)
  - Heartbeat: Server sends ping every 30s (client responds pong)
  - Idle timeout: Close connection if no activity for 5 minutes
  - Reconnect logic: Client exponential backoff (3s, 6s, 12s, max 60s)

- [x] **[MEDIUM]** Integration with existing ExecutionHistoryService API
  - Reuse `queryExecutionHistory()` API for delta detection (query recent changes)
  - Delta: Compare current snapshot with previous snapshot (last 10 minutes)
  - Send only fields that changed (execution status, duration, error_context, updated_at)

### WebSocket Server Architecture

```typescript
// WS Server Setup
const wss = new WebSocketServer({ port: 9001 });

// Connection lifecycle
wss.on('connection', (ws, req) => {
  const userId = authenticateUser(req); // Extract from JWT token
  const channel = `execution-updates:${userId}`;

  // Subscribe to Redis pub/sub
  redisSubscriber.subscribe(channel, (err) => {
    if (err) ws.close(1011, 'Subscribe failed');
  });

  // Send heartbeat every 30s
  const heartbeat = setInterval(() => ws.ping(), 30000);

  // Handle messages & disconnects
  ws.on('close', () => {
    clearInterval(heartbeat);
    redisSubscriber.unsubscribe(channel);
  });
});

// Broadcast delta updates
app.post('/api/automations/execute', async (req, res) => {
  // 1. Execute automation
  const result = await executeAutomation(...);

  // 2. Publish delta to Redis
  const delta = { execution_id, status, duration, updated_at };
  await redisPublisher.publish(`execution-updates:${userId}`, JSON.stringify(delta));

  // 3. Return response
  res.json(result);
});
```

### Delta Detection Algorithm
```typescript
// Simplified delta detection
async function detectDeltas(userId: string, since?: Date): Promise<Delta[]> {
  const recent = await queryExecutionHistory({
    userId,
    startDate: since || new Date(Date.now() - 10 * 60000), // Last 10 minutes
    endDate: new Date(),
    limit: 100
  });

  const deltas: Delta[] = [];
  for (const exec of recent) {
    const changed = {};
    if (exec.status !== cache.get(`exec:${exec.id}:status`)) {
      changed.status = exec.status;
    }
    if (exec.duration !== cache.get(`exec:${exec.id}:duration`)) {
      changed.duration = exec.duration;
    }
    if (JSON.stringify(exec.error_context) !== cache.get(`exec:${exec.id}:error`)) {
      changed.error_context = exec.error_context;
    }

    if (Object.keys(changed).length > 0) {
      deltas.push({
        execution_id: exec.id,
        changes: changed,
        updated_at: exec.updated_at
      });
      // Update cache
      Object.entries(changed).forEach(([key, val]) => {
        cache.set(`exec:${exec.id}:${key}`, val);
      });
    }
  }
  return deltas;
}
```
