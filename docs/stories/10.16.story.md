## Dev Notes

### Relevant Context

- O frontend já possui um primitive `Breadcrumb`, mas ainda sem resolução dinâmica conectada ao shell.
- As rotas dinâmicas mais relevantes desta trilha são `cards`, `webhooks` e `automations`.
- O nome exibido deve ser resolvido a partir dos dados já carregados na página ou de serviços existentes, sem criar backend dedicado só para breadcrumb.

### Roteiro de Breadcrumbs

| Rota | Breadcrumbs |
|------|-------------|
| `/` | Dashboard |
| `/dashboard` | Dashboard |
| `/dashboard/automations` | Dashboard > Fluxos |
| `/dashboard/automations/[id]` | Dashboard > Fluxos > [Nome do Fluxo] |
| `/dashboard/webhooks/[id]` | Dashboard > Webhooks > [Nome do Webhook] |
| `/cards/[id]` | Dashboard > Cards > [Nome do Card] |
| `/auth/login` | Login |

### Resolução Dinâmica

```tsx
// utils/breadcrumbs.ts
interface BreadcrumbItem {
  label: string;
  href?: string;
  isDynamic?: boolean;
}

export function useBreadcrumbs(): BreadcrumbItem[] {
  const router = useRouter();
  const pathname = router.pathname;

  // Static routes
  const staticMap: Record<string, BreadcrumbItem[]> = {
    '/': [{ label: 'Dashboard' }],
    '/dashboard': [{ label: 'Dashboard' }],
    '/dashboard/automations': [
      { label: 'Dashboard', href: '/' },
      { label: 'Fluxos' },
    ],
  };
```
