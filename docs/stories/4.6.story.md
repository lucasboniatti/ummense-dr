## Story

**As a** TaskFlow user,
**I want** to click on a card and see its details with all tasks in a modal,
**so that** I can view, create, edit, and complete tasks within a client/card.

---

## Acceptance Criteria

1. [x] `CardDetailModal` component created at `packages/frontend/src/components/CardDetailModal.tsx`
2. [x] Modal opens when clicking a card in Kanban or List view
3. [x] Modal shows card info: title (editable), description (editable), current column name, flow name
4. [x] Modal displays all tasks for the card using `useTasks(cardId)` hook
5. [x] Tasks show: title, priority badge (P1=red, P2=yellow, P3=gray), status, due date
6. [x] Checkbox to toggle task status between 'open' and 'done' (calls `PUT /api/tasks/:id`)
7. [x] "Add Task" form at bottom: title (required), priority (select P1/P2/P3), due date (optional)
8. [x] Task inline edit: click to edit title, priority, due date
9. [x] Delete task with confirmation
10. [x] Card title/description editable inline (calls `PUT /api/cards/:id`)
11. [x] Modal closable via X button, Escape key, or backdrop click
