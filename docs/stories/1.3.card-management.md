## Story
Como usuário, preciso gerenciar cards que representam clientes/projetos, com descrição, comentários e lista de tarefas associadas.

## Acceptance Criteria

### AC-1: Card Display
- [ ] Mostra descrição, cliente link, comentários

- [ ] Clicável para expandir/editar

### AC-2: Card Operations
- [ ] POST /api/cards - Criar

### AC-3: Comments
- [ ] Adicionar comentários
- [ ] Editar/deletar próprios comentários
- [ ] Timestamp automático

### AC-4: Database
- [ ] Tabela `cards` (title, description, column_id, user_id)
- [ ] Tabela `comments` (text, card_id, user_id)
