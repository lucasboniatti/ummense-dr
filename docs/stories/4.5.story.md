## Story

As an **operator**, I want to **validate required environment configuration before startup**, so that **local runs fail fast and avoid hidden runtime defects**.

## Acceptance Criteria

- [x] Missing mandatory variables block startup with explicit diagnostics.
- [x] Default insecure placeholders are removed from runtime-critical paths.
- [x] Environment troubleshooting guide is published.
