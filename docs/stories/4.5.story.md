## Story

**As a** TaskFlow user,
**I want** the Kanban board connected to real data with working drag-and-drop,
**so that** I can see my flows with cards in columns and move cards between columns.

---

## Acceptance Criteria

1. [x] KanbanBoard component refactored to use `useCards(flowId)` hook instead of local mock data
2. [x] Board loads columns and cards from `GET /api/flows/:flowId` on mount
3. [x] Cards display title, description preview, and task summary badge (e.g., "3/5 done")
4. [x] Drag-and-drop moves cards between columns with optimistic UI update
5. [x] On drop: calls `PUT /api/cards/reorder` to persist new positions
6. [x] On drag error: reverts to previous state and shows error toast
7. [x] "Add Card" button per column creates card via `POST /api/cards`
8. [x] Card click opens CardDetailModal (Story 4.6 dependency — stub/empty modal OK for now)
9. [x] Loading skeleton shown while data is fetching
10. [x] Empty state shown when flow has no cards

### Existing Components to Refactor

- `Card.tsx` — Props: title, description, tags[], draggable, onDragStart, onClick
- `Column.tsx` — Props: name, cardCount, onDragOver, onDrop, children

### Existing Drag-Drop Pattern
[Source: packages/frontend/src/components/KanbanBoard.tsx]
- Uses HTML5 drag-and-drop API (onDragStart, onDragOver, onDrop)
- `useKanban` hook manages draggedCard state
- Adapt to call `cardsApi.reorder()` on drop

### Optimistic Update Pattern
[Source: docs/architecture/taskflow-architecture.md#5.5]
1. User drops card → Update local state immediately
2. Call `cardsApi.reorder()` in background
3. On error → Revert local state + show toast

### Error Toast
- Use a simple toast/notification component (create if none exists in project)
- Pattern: `showToast({ message: "Erro ao mover card", type: "error" })`
- Can be a minimal div with absolute positioning + setTimeout auto-dismiss

### Testing

- Test Kanban renders columns and cards from API data
- Test drag-drop updates card positions
- Test optimistic revert on API error
- Test "Add Card" creates card and appears in column
- Test loading/empty states
