## Technical Notes

### URL Validation Rules (Client-Side)
```javascript
const urlValidation = (url) => {
  // 1. Must be HTTPS
  if (!url.startsWith('https://')) return false;

  // 2. Valid URL format
  try { new URL(url); } catch { return false; }

  // 3. Not localhost or private IP (SSRF check)
  const hostname = new URL(url).hostname;
  const blocklist = ['localhost', '127.0.0.1', ...];
  if (blocklist.includes(hostname)) return false;

  return true;
};
```

### API Key Display Pattern
```javascript
// Generation
const apiKey = generateRandomKey(32); // Full key
const displayKey = `sk_${apiKey.slice(-4)}`;

// Storage
user_webhook.api_key_hash = bcrypt.hash(apiKey);
user_webhook.api_key_preview = displayKey;

// Display in UI
<input value={displayKey} disabled /> // Only show last 4
```

### Delivery History Query Performance
```sql
-- Indexed for fast filtering
SELECT * FROM webhook_deliveries
WHERE webhook_id = $1
AND created_at BETWEEN $2 AND $3
AND status IN ('success', 'failed', 'dead_lettered', 'pending')
ORDER BY created_at DESC
LIMIT 50;

-- Indexes:
CREATE INDEX idx_webhook_deliveries_webhook_created
ON webhook_deliveries(webhook_id, created_at DESC);
CREATE INDEX idx_webhook_deliveries_status
ON webhook_deliveries(webhook_id, status);
```
