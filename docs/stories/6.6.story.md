## API Contract Baseline (Mandatory)

- GET `/api/panel/overview`
- GET `/api/flows/:flowId`
- PATCH `/api/cards/:cardId/move`
- GET `/api/cards/:cardId`
- GET `/api/cards/:cardId/timeline`
- POST `/api/cards/:cardId/timeline`
- GET `/api/tasks/:taskId/history`
- GET `/api/events`
- POST `/api/events`
- PATCH `/api/events/:eventId`
- DELETE `/api/events/:eventId`

Contract rules:
- Todos endpoints autenticados (Bearer JWT).
- Erros padronizados: 400, 401, 403, 404, 409, 422, 500.
- Paginação obrigatória em listagens (`limit`, `offset`).
- Formato de data/hora: ISO-8601 UTC.
