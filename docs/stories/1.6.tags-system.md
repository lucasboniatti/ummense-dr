# Story 1.6: Sistema de Tags Customizáveis

**Epic:** 1.0
**Status:** Ready for Review
**Priority:** P1
**Points:** 5

## Story
Como usuário, preciso criar tags customizáveis com cores que posso aplicar em cards e tarefas para categorização.

## Acceptance Criteria

### AC-1: Tag Management
- [ ] POST /api/tags - Criar tag com nome e cor
- [ ] GET /api/tags - Listar tags do usuário
- [ ] PUT /api/tags/:id - Editar
- [ ] DELETE /api/tags/:id - Deletar

### AC-2: Tag Application
- [ ] Aplicar tag em cards
- [ ] Aplicar tag em tarefas
- [ ] Remover tags
- [ ] Filtrar por tag

### AC-3: Tag Display
- [ ] Mostrar tag como badge colorido
- [ ] Nome da tag como label
- [ ] Cor customizável (color picker)

### AC-4: Database
- [ ] Tabela `tags` (name, color, user_id)
- [ ] Tabela `card_tags` (card_id, tag_id)
- [ ] Tabela `task_tags` (task_id, tag_id)
