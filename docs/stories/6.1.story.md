## Story

As an **operations user**, I want to **acessar um painel consolidado com navegação lateral, topo global, tarefas e visão temporal**, so that **eu consiga operar o dia de trabalho em uma única superfície de produto**.

---

## Acceptance Criteria

- [x] Estrutura global com sidebar fixa (Painel, Fluxos, Contatos, Arquivos, Mais) e topbar funcional (busca, filtro, notificações, ações rápidas).
- [x] Painel principal renderiza lista de tarefas com progresso, prioridade, responsável, tags, vínculo de card e prazo.
- [x] Painel exibe bloco temporal lateral com calendário mensal e resumo de eventos.
- [x] A experiência é responsiva (desktop/tablet/mobile) mantendo usabilidade operacional.
- [x] Estado de loading/erro/vazio é tratado de forma explícita para todos os widgets do painel.
- [x] Navegação não quebra sessões existentes de webhooks/automations já entregues.

---

## Tasks / Subtasks

- [x] Implementar layout base e navegação global no frontend (AC: 1, 4).
