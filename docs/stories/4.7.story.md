## Story

**As a** TaskFlow user,
**I want** a sidebar to navigate between flows and toggle between Kanban and List views,
**so that** I can manage multiple flows and choose the best visualization for my workflow.

---

## Acceptance Criteria

1. [x] `FlowSidebar` component shows all user's flows as a list
2. [x] Clicking a flow in sidebar navigates to that flow's page
3. [x] "New Flow" button in sidebar creates a flow (name input) and navigates to it
4. [x] `ListView` component shows all cards in a linear table format
5. [x] List view columns: Card title, Column name, Task count (done/total), Last updated
6. [x] View toggle button switches between Kanban and List views (persists in URL or local state)
7. [x] `FlowHeader` component shows flow name, description, and view toggle
8. [x] Flow page route: `/flows/[flowId]` renders either Kanban or List view
9. [x] Default flow page: `/flows` shows first flow or "Create your first flow" empty state
10. [x] Clicking a card row in List view opens CardDetailModal
